Class ExpressionDB::EnsemblGene
In: lib/files/activerecord.rb
Parent: DBConnection

DESCRIPTION

A table holding information on EnsEMBL genes. Genes belong to genome_dbs and are connect to samples through xref_samples.

USAGE

     gene = ExpressionDB::EnsemblGene.find_by_stable_id('ENSG00000121101')
     gene.xref_samples.each do |xref|
             puts "#{x.sample.name} #{x.fpkm}"
     end
